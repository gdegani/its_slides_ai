# Images per le slide

Questa cartella contiene le immagini utilizzate nelle slide.

## Screenshot LM Studio necessari

Per completare la slide "LM Studio: eseguire LLM in locale", sono necessari i seguenti screenshot:

### 1. lmstudio-models.png
Screenshot della lista dei modelli LLM disponibili in LM Studio.
- Apri LM Studio
- Vai alla sezione "Discover" o "Models"
- Fai uno screenshot che mostri l'elenco dei modelli disponibili per il download
- Salva come `lmstudio-models.png`

### 2. lmstudio-chat.png
Screenshot di un esempio di interazione con la chat di LM Studio.
- Apri LM Studio
- Carica un modello (es. Llama, Mistral, o simile)
- Vai alla sezione "Chat"
- Fai una domanda di esempio relativa alla programmazione in C
- Fai uno screenshot che mostri la conversazione
- Salva come `lmstudio-chat.png`

## Note
- Formato consigliato: PNG
- Risoluzione: almeno 1920x1080 per buona qualità
- Dimensione file: cerca di mantenere sotto 1MB per screenshot
